# Security policy

## Reporting a vulnerability

Report security vulnerabilities through GitHub's [private vulnerability
reporting](https://github.com/ss0923/limb.tmux/security/advisories/new).
Do not open a public issue.

A maintainer will acknowledge receipt within seven days and provide an
initial assessment within fourteen days.

## Supported versions

Security fixes are applied to the `main` branch. Tagged releases are
supported until superseded by a later tag on the same minor line.
